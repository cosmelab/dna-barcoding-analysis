ALIGNMENT SCRIPTS (scripts/ directory):
---------------------------------------

1. run_mafft.sh
   Type: Bash shell script
   Size: ~10 KB, 308 lines
   Purpose: MAFFT sequence alignment wrapper
   Key Functions:
     - Multiple alignment strategy selection
     - Parallel processing support
     - Input validation and error handling
     - Colored progress reporting
   Dependencies: MAFFT, bash 4.0+
   
   Main Entry Point: usage() function
   Key Functions:
     - build_mafft_command(): Constructs MAFFT command with parameters
     - check_mafft_installed(): Validates MAFFT availability
     - get_sequence_count(): Counts sequences in FASTA file

2. alignment_stats.py
   Type: Python script (executable)
   Size: ~14 KB, 401 lines
   Purpose: Calculate alignment quality metrics
   Key Class: AlignmentStats
   Methods:
     - load_alignment(): Load from file
     - get_gap_percentage(): Overall and per-position gaps
     - calculate_shannon_entropy(): Sequence variation measure
     - calculate_conservation_score(): Conservation index (0-1)
     - calculate_identity(): Sequence identity percentage
     - export_position_statistics_csv(): CSV export
     - print_summary(): Display statistics
   Dependencies: BioPython, NumPy
   Output: Console display + optional CSV file

3. visualize_alignment.py
   Type: Python script (executable)
   Size: ~15 KB, 418 lines
   Purpose: Create publication-quality alignment visualizations
   Key Class: AlignmentVisualizer
   Methods:
     - plot_alignment_matrix(): Colored sequence matrix
     - plot_conservation_profile(): Line/fill conservation plot
     - plot_gap_distribution(): Gap percentage histogram
     - create_full_visualization(): Complete 3-panel figure
     - show_plot(): Interactive display
   Output Formats: PNG, PDF, SVG, JPG (via matplotlib)
   Dependencies: BioPython, NumPy, Matplotlib

4. trim_alignment.py
   Type: Python script (executable)
   Size: ~19 KB, 514 lines
   Purpose: Remove poorly aligned regions
   Key Class: AlignmentTrimmer
   Methods:
     - trim_by_gap_threshold(): Remove high-gap positions
     - trim_by_conservation(): Remove low-conservation regions
     - trim_sequences_by_gap(): Remove gap-rich sequences
     - apply_preset_strategy(): Use strict/moderate/lenient
     - create_trimmed_alignment(): Generate output alignment
     - print_trimming_report(): Summary statistics
   Output: Trimmed FASTA file + report
   Dependencies: BioPython, NumPy

USAGE SUMMARY
================================================================================

Typical 4-Script Workflow:

1. Align sequences:
   cd 06_alignment/scripts/
   ./run_mafft.sh -i unaligned.fasta -o aligned.fasta

2. Calculate statistics:
   ./alignment_stats.py -i aligned.fasta -o stats.csv

3. Visualize alignment:
   ./visualize_alignment.py -i aligned.fasta -o viz.pdf

4. Trim poorly aligned regions:
   ./trim_alignment.py -i aligned.fasta -o trimmed.fasta

See EXAMPLES.md for 10 additional real-world scenarios.
