- `pcr_products_raw.csv` - Individual PCR measurements
- `pcr_products_summary.csv` - Mean ¬± StdDev by sample type

- `wolbachia_pcr.csv` - Wolbachia detection results
- `coi_pcr_gel_results.csv` - Gel band visibility
